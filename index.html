FTP Anonymous Login — Servidor Público

002

MySQL 5.7 Exposto — Porta 3306 Pública

Banco de dados acessível externamente em versão EOL (End of Life desde 2023). Combinação de exposição pública com versão sem suporte representa risco significativo de exploração via CVEs conhecidos.

MySQL EOL Software Network Exposure
High

Enumeração SNMP — Community String Padrão

Dispositivos de rede respondendo a queries SNMP com community string "public". Possível extração de topologia de rede, tabelas ARP, informações de interfaces e configurações de roteamento.

SNMP Network Information Disclosure
High
